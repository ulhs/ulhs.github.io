Data Privacy Notice

In compliance with the Philippines Data Privacy Act of 2012 (RA 10173), Upper Labay High School ensures that all student data processed by this Attendance System is handled with the highest level of security. This system operates on a "Zero-Knowledge" architecture:

Local Processing

Data is processed entirely within the local browser memory. Student names, LRNs, and photos are never uploaded to servers or stored online.

Session-Only Storage

Registry data and photos exist only for this active session. Closing the browser tab immediately wipes all sensitive information from this terminal.

Purpose Limitation

Collected data is used strictly for official DepEd SF2 Attendance Reporting. No personal information is shared with third parties or used for unauthorized profiling.
